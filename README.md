# Wedding Website

React + TypeScript + Vite starter for building a wedding website (Zola-inspired, custom-built).

## Run locally

```bash
npm install
npm run dev
```

## Wedding page implementation checklist

- [ ] Home / hero section (names, date, location, quick CTA)
- [ ] Our story timeline
- [ ] Schedule of events (welcome party, ceremony, reception, after party)
- [ ] Venue details with maps, parking, and accessibility info
- [ ] Travel and accommodations (hotel blocks, airports, transit)
- [ ] RSVP flow (attending, guest count, meal preferences, plus-ones)
- [ ] Registry links
- [ ] FAQ (dress code, kids policy, weather, contact)
- [ ] Wedding party section
- [ ] Photo gallery / engagement photos
- [ ] Countdown timer
- [ ] Contact form or point-of-contact section
- [ ] Mobile responsive design and accessibility checks
